# Sistema de Login — React

Versão React completa do sistema de login acadêmico.

## Páginas incluídas

- `/login` — Tela de login
- `/cadastro` — Cadastro (passo 1 com CPF)
- `/cadastro-finalizado` — Confirmação de cadastro
- `/dashboard` — Painel do aluno

## Como rodar

**Pré-requisito:** Node.js instalado (versão 16 ou superior)

```bash
# 1. Instalar dependências
npm install

# 2. Rodar em modo desenvolvimento
npm run dev
```

Acesse: http://localhost:5173

## Tecnologias

- React 18
- React Router DOM 6
- Vite 5
